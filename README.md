# VIRALO — лендинг AI-видео креативов

Одностраничный сайт для услуги: клиент присылает голосовое сообщение или короткое ТЗ, а получает готовый видео-креатив для Reels, Shorts, TikTok, Telegram, сторис, рекламы или контент-плана.

## Оффер

- 1 видео — 9 900 ₽
- 10 видео — 49 900 ₽
- 25 видео — 100 000 ₽

Правки в услугу не входят: результат создаётся по исходному ТЗ. Если нужна другая идея, акцент или подача, это оформляется как новое видео.

## Стек

- Next.js 15 App Router
- React 18
- TypeScript
- Tailwind CSS 3
- Статический экспорт для GitHub Pages

## Команды

```bash
npm install
npm run dev
npm run build
```

Локально сайт открывается на `http://localhost:3000`.

## Деплой

Проект настроен на GitHub Pages через `.github/workflows/deploy.yml`.

Для GitHub Pages используется:

```bash
GITHUB_PAGES=true npm run build
```

Это создаёт статическую версию сайта в папке `out`.

## Важные файлы

- `lib/content.ts` — тексты и данные лендинга
- `app/page.tsx` — порядок секций
- `app/globals.css` — визуальный стиль, токены, анимации
- `components/sections/` — секции страницы
